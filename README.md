# Nextrap Colors



## SCSS Variables

To edit the color schema of the Nextrap theme, you can edit the following SCSS variables:

```scss
$primary: #000000;
$accent: #000000;
$secondary: #000000;
$danger: #000000;
$warning: #000000;
$success: #000000;
```

## CSS Variables

```css
--primary-rgb: 0, 0, 0;
```
